람다식 : 익명구현객체를 식으로 표현

람다식으로 사용할 수 있는 타겟타입(인터페이스)
- 추상메소드를 하나만 가지고 있어야 함
- @FunctionalInterface 컴파일 시 오류 검출 => 추상메서드가 하나인지 검증

표준 API 의 함수적 인터페이스
1) consumer : 매개값은 있고, 리턴값 없음
   void accpt(T t) : 받아서 소비

2) supplier : 매개값은 없고 리턴값은 있음
    T get() : 공급

3) Function : 매개값도 있고, 리턴값도 있음 / 주로 매개값을 리턴값으로 매핑
4) Operator : 매개값도 있고, 리턴값도 있음 / 주로 매개값을 연산하고 결과를 리턴
5) Perdicate : 매개값은 있고, 리턴값은 boolean / 매개값을 조사해서 t/f 리턴


- 메서드 참조
하나의 메서드만 호출하는 람다식은 클래스이름::메서드이름 or 참조변수::메서드이름

System.out::print
String::toUpperCase
